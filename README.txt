# FreeComplex

Uma versão livre e totalmente baseada em shell scripting do MuComplex.

Estrutura:
.salaX - Cada diretório é um desafio.
.salaX/.motd - Texto de abertura do desafio. (Eventualmente podemos usar pygments ou outro para deixar isso mais bonito.)
.salaX/.env - Script que é parseado quando o usuário entar na sala.
